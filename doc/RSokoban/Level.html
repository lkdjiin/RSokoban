RSokoban::Level

Je suis un niveau du jeu. Pour réussir un niveau, placer chaque caisses (’$’) sur un emplacement de stockage (’.’).

new(rawLevel) click to toggle source

Je construit le niveau à partir d’un format brut. @example un niveau en format brut 

      'Level 1', ['#####', '#.o@#', '#####']

  • ’#’ est un mur

  • ’.’ est un emplacement de stockage

  • ’$’ est une caisse

  • ’@’ est le bonhomme

  • ’ ’ est un bout de sol vide
