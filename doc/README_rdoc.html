Welcome to RSokoban !

RSokoban is a clone of the famous Sokoban game. I wrote this program just to improve my skills in Ruby. Currently, you can play only in a console window. 



      #####
      #   #
      #$  #

  #   # ## #####  ..#
  # $  $          ..#
  ##### ### #@##  ..#
      #     #########
      #######
Enjoy the game.

Dependancies

Ruby >= 1.9.2

Installing RSokoban

Installing via a gem is planned. Currently, just copy the files somewhere, go to the bin/ folder and type : 

        ./rsokoban

to start playing.

To run tests, go to the bin/ folder and type : 

  ./test.rb
