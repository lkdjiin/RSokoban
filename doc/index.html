Welcome to RSokoban !

RSokoban is a clone of the famous Sokoban game. I wrote this program just to improve my skills in Ruby. Currently, you can play only in a console window.

Dependancies

Ruby >= 1.9

Installing RSokoban

Installing via a gem is easy. Go to the RSokoban folder and build the gem : 

 gem build rsokoban.gemspec

Then install it : 

 gem install RSokoban-x-xx.gem

Now start playing : 

 rsokoban
